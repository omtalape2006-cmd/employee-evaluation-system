# 🤖 AI HR Expert System

An intelligent employee evaluation and HR analytics web application built with **Python**, **Streamlit**, and **SQLite**.

## 🚀 Live Features

* 📊 Employee Performance Evaluation
* 🧠 AI-Based HR Diagnosis
* 📈 KPI, Quality & Attendance Scoring
* ⭐ Promotion / Risk Detection
* 🗑️ Delete Employee Records
* 💬 Built-in HR Chatbot
* 💾 SQLite Database Support
* ☁️ Deployable on Streamlit Cloud

---

## 🛠️ Tech Stack

* Python
* Streamlit
* SQLite
* Pandas

---

## 📂 Project Files

* `app.py` → Main Streamlit Application
* `database_setup.py` → Creates Database & Table
* `hr_expert_system.db` → SQLite Database
* `requirements.txt` → Required Packages

---

## ▶️ Run Locally

```bash
pip install -r requirements.txt
python database_setup.py
streamlit run app.py
```

---

## 🌐 Deployment

Deploy easily using **Streamlit Community Cloud**.

1. Upload project to GitHub
2. Login to Streamlit Cloud
3. Select repository
4. Choose `app.py`
5. Deploy 🚀
